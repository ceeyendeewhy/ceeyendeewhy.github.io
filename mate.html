An android application for room/living/house mates

Signing up for Mate is easy. Just register with your name, email, and password. The registration system is implemented using MySQL, PHP, and SQLite.

Once you register and log in, you will be prompted to join a new home or an existing home.

Add a new home if your mates have not done so already.

Members of the same home can view a shared calendar that displays events with event times and the user who created the event.

Double tap to add a new event.

And pay bills and fees through PayPal!
