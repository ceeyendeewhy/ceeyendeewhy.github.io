Gitlet

A version control system incorporating core features of git and more using serialization and Java File classes.

View more projects

Begin by initializing Gitlet. Then add files and view status.

The log and global-log features allow user to view commits.

You can remove files so they do not show up in the next commit.

Create new branches and checkout to those branches. Other features of Gitlet include find, remove branch, reset, merge, rebase, and interactive rebase.
